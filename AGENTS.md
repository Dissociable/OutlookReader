# AI Agent Directives

The following rules apply to all AI agents working within this repository:

1. **Package Manager**: Always use `pnpm` wherever possible for installing dependencies, running scripts, and managing the project. Do not use `npm` or `yarn` or `npx` (unless using `pnpm dlx`).
2. **Framework**: React + TypeScript + Vite.
3. **Styling**: Tailwind CSS v4+ (Use `@tailwindcss/vite`, no `tailwind.config.js` or `postcss.config.js`) + shadcn/ui.
